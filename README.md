# rust-minicat

```
A lesser cat utility written in rust. Concatenate and print files

Usage: minicat [OPTIONS] [FILES]...

Arguments:
  [FILES]...  Number the output lines, starting at 1 [default: -]

Options:
  -n, --number-lines           
  -b, --number-nonblank-lines  Number the non-blank output lines, starting at 1
  -h, --help                   Print help
  -V, --version                Print version
```
